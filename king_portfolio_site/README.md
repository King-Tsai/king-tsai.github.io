# KING Portfolio Site

## 結構
- `index.html`：網站主頁
- `assets/css/styles.css`：樣式
- `assets/js/data.js`：網站內容資料（可直接編輯）
- `assets/js/main.js`：前台渲染腳本
- `assets/images/certs/`：專業認證圖片
- `assets/images/education/`：學歷與學習表現圖片

## 如何修改內容
1. 開啟 `assets/js/data.js`
2. 直接修改其中欄位
3. 重新整理 `index.html` 即可看到更新

## 可調整重點
- `profile`：姓名、標題、摘要、座右銘
- `contact`：聯絡 Email 與 mailto 預設內容
- `skillCategories`：技術與工具分類
- `experiences`：工作經歷
- `education`：學歷
- `certifications`：專業認證（可替換圖片與標題）
- `projects`：專案成果
